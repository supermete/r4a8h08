# Installation

## Stable release

To install r4a8h08, run this command in your terminal:

```sh
uv add r4a8h08
```

Or if you prefer to use `pip`:

```sh
pip install r4a8h08
```

## From source

The source files for r4a8h08 can be downloaded from the [Github repo](https://github.com/supermete/r4a8h08).

You can either clone the public repository:

```sh
git clone https://github.com/supermete/r4a8h08
```

Or download the [tarball](https://github.com/supermete/r4a8h08/tarball/main):

```sh
curl -OJL https://github.com/supermete/r4a8h08/tarball/main
```

Once you have a copy of the source, you can install it with:

```sh
cd r4a8h08
uv sync
```
